# Multiomics Python Course - Homework Portfolio

This repository contains the short hands-on homework exercises from a Python data analysis course.
Each class introduced one analysis concept in a lecture and then ended with a compact practical task.

This repository reflects **course exercises**, so notebooks are concise and focused on core concepts rather than production pipelines. Some notebooks embed lecture slide screenshots directly in markdown cells.

The exercises are organized into topic folders:

- `Regression`
- `PCA`
- `Clustering`
- `Classifier`
- `Statistical-testing`
- `time-series-analysis`

Most tasks use small biological datasets (proteomics, lipidomics, cardiometabolic markers) and common scientific Python tools such as **pandas**, **NumPy**, **matplotlib**, **scikit-learn**, and **SciPy**.

---

## 1) Regression (`Regression/linear-nonlinear-regression.ipynb`)

Goal: Practice both linear and nonlinear curve fitting.
Concepts practiced: Linear regression, residual-based fitting, nonlinear optimization, model visualization and extrapolation.

---

## 2) Principal Component Analysis (`PCA/principal-component-analysis.ipynb`)

Goal: Reduce dimensionality and interpret variance in high-dimensional proteomics data.
Concepts practiced: Feature scaling, unsupervised dimensionality reduction, variance interpretation and PCA scatter plots.

---

## 3) Clustering (`Clustering/Clustering.ipynb`)

Goal: Group samples into unsupervised clusters and visualize cluster structure.
Concepts practiced:  Unsupervised learning, cluster assignment, preprocessing for distance-based methods and cluster visualization.

---

## 4) Classification (`Classifier/Classifier.ipynb`)

Goal: Compare multiple supervised classification algorithms on a tabular health dataset.
Concepts practiced: Supervised learning workflow, model comparison, cross-validation and inference on new observations.

---

## 5) Statistical testing (`Statistical-testing/Statistical-testing.ipynb`)

Goal: Identify significant lipid changes between two conditions/time points.
Concepts practiced: Hypothesis testing in high-dimensional data, unequal-variance t-tests, false discovery control, and interpretation of significance results.

---

## 6) Time-series analysis (`time-series-analysis/series-analysis.ipynb`)

Goal: Summarize repeated measurements over time and discover temporal profile groups.
Concepts practiced: Time-course aggregation, profile-based clustering, cosine similarity, and hierarchical clustering interpretation.
